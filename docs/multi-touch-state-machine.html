3 Action 事件 × 状态 转移表

Action 事件 INACTIVE WAITING ACTIVE BLOCKING
ACTION_DOWN → WAITING （不消费） → WAITING（新序列） → WAITING（新序列） → WAITING（同 INACTIVE）
ACTION_POINTER_DOWN 不动 （不消费） 达阈值→ACTIVE （消费）；否则留 WAITING 留 ACTIVE （消费） 留 BLOCKING （劫持抛弃）
ACTION_MOVE 不动 （不消费） 同步坐标；超时→INACTIVE 留 ACTIVE （消费） 留 BLOCKING （劫持抛弃）
ACTION_POINTER_UP 不动 （不消费） 该指入 ignoredIds；留 WAITING → 判定 → BLOCKING/INACTIVE （消费） 留 BLOCKING （劫持抛弃）
ACTION_UP → INACTIVE （不消费） → INACTIVE （不消费） → 判定 → BLOCKING/INACTIVE （消费） 留 BLOCKING （劫持抛弃）
ACTION_CANCEL 任意状态 → reset() → INACTIVE（不消费，交系统处理）
